The Recipe Discovery App is built with React, TypeScript, React Router v6, and Context API to interact with TheMealDB API. It features custom useFetch and useLocalStorage hooks, search, and a favorites system, allowing users to browse meals by category, view details, and save favorites

To use this project, first create a new react project and to install dependencies for the project, you’ll need to run from the project the command npm install. And to run the server, you’ll need to run the command npm run dev

The project is structured in modular directories to provide a clean, loose coupled and maintainable architecture.
